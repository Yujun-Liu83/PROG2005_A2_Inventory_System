Help & FAQ

How do I add a new item?

Go to the Inventory page, fill in the "Add New Item" form (all fields except comments are required), then click "Add Item".

How do I edit an existing item?

On the Inventory page, scroll down to "Edit Item by Name". Enter the current item name, click "Load Data", modify the fields, and click "Save Changes". Note: The item name is used as the unique identifier; you can also change the name.

How do I delete an item?

On the Inventory page, each row in the "All Items" table has a "Delete" button. Click it and confirm the deletion.

Can I search for items?

Yes, use the Search page. You can search by item name (partial match) and filter by category and stock status.

What are "Popular Items"?

Popular items are those marked with "Popular" checkbox when adding or editing. They appear on the Popular page.

Is my data saved permanently?

No, all data is stored in the browser's memory only. Refreshing the page or closing the tab will reset the data to the default sample items. This meets the assignment requirement of no server-side persistence.

Troubleshooting: The page looks broken on mobile?

The application uses responsive design. If the layout appears misaligned, try rotating your device to landscape or ensure you have the latest browser version. The tables will convert to block display on narrow screens.
