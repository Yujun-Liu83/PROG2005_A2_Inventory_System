Data Collection & Use

This application does not collect any personally identifiable information. All inventory data you enter is stored exclusively in your browser's memory during the current session. No data is transmitted to any server, and we do not log any usage statistics.

  • ✅ No personal information (name, email, IP) is collected
  • ✅ No analytics or performance monitoring
  • ✅ No cookies or local storage for persistent data

Data Storage & Retention

Data is stored ephemerally in your browser's memory. Refreshing the page or closing the tab will reset the data to the default sample set. There is no permanent database or file storage.

➡️ User adds/edits items → Stored in memory
➡️ Browser tab closed → Data cleared
➡️ No backup or recovery

Security Measures

We implement industry-standard practices to protect your data within the browser environment:

  • ✔️ Angular's DOM sanitization (prevents XSS)
  • ✔️ Input validation (type & range checks)
  • ✔️ No external scripts or iframes
  • ✔️ Content Security Policy (CSP) ready

Your Rights

Because we collect no personal data, there is no personal data to access, rectify, or delete. However, you have full control over the inventory data you input: you can add, edit, or delete any item at any time within your session.

Third-Party Sharing & Links

We do not share any data with third parties. This application contains no external links, ads, or social media plugins. All content is self-contained.
